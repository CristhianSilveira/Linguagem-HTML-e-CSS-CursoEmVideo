------ TAGS HTML5 ------

1- <!DOCTYPE html>: define a versão HTML do documento.

2- <html>: define o início e o fim do documento HTML.

3- <head>: define uma seção de cabeçalho do documento HTML, que inclui informações como título, links para folhas de estilo CSS e meta-informações.

4- <title>: define o título do documento, que aparece na aba do navegador.

5- <body>: define uma seção do corpo do documento HTML, que inclui o conteúdo visível do documento.

6- <header>: define o cabeçalho do documento, que geralmente inclui informações como logotipo, menus de navegação e outros elementos que aparecem no topo da página.

7- <nav>: define uma seção de navegação do documento, que geralmente inclui links para outras páginas do site.

8- <section>: define uma seção do documento, geralmente com um tema ou propósito específico.

9- <article>: define uma seção de conteúdo independente e autossuficiente do restante do documento.

10- <aside>: define uma seção lateral do documento, que geralmente inclui informações complementares ou de suporte ao conteúdo principal.

11- <footer>: define o rodapé do documento, que geralmente inclui informações como direitos autorais, links para contatos e outros elementos que aparecem na parte inferior da página.

12- <p>: define um parágrafo de texto.

13- <a>: defina um link para outra página ou recurso.

14- <img>: define uma imagem que será exibida no documento.

15- <ul>: define uma lista não ordenada de itens.

16- <ol>: define uma lista ordenada de itens.

17- <li>: define um item em uma lista.

18- <table>: define uma tabela que exibe informações em linhas e sonoras.

----- TAGS CSS ----

1- color: define a cor do texto dentro de um elemento.
2- background-color: define um cor do fundo dentro de um elemento.
3- font-size: define o tamanho da fonte dentro de um elemento.
4- font-family: define a família de fontes que será usada para o texto dentro de um elemento.
5- font-weight: define o peso (negrito, normal, mais claro, etc.) da fonte dentro de um elemento.
6- text-align: alinha o texto dentro de um elemento (esquerda, centro, direita, justificar).
7- text-decoration: define as decorações para o texto dentro de um elemento (sublinhado, line-through, nenhum, etc.).
8- padding: define o espaço entre o conteúdo e a borda de um elemento.
9- margin: define o espaço ao redor de um elemento.
10- border: define a borda de um elemento (largura, estilo e cor).
11- display: define como um elemento deve ser exibido (block, inline, none, etc.).
12- position: define a posição de um elemento em relação ao seu contêiner (estático, relativo, absoluto, fixo).
13 -float: define como um elemento deve se comportar em relação a outros elementos em torno dele.
14- width: define a largura de um elemento.
15 -height: define a altura de um elemento.
16- text-transform: define uma transformação de texto dentro de um elemento (maiúsculas, minúsculas, maiúsculas, nenhum).
17- line-height: define a altura da linha dentro de um elemento.
18- text-indent: define um recuo da primeira linha de texto dentro de um elemento.
19- word-spacing: define o espaçamento entre as palavras dentro de um elemento.
20- letter-spacing: define o espaçamento entre as letras dentro de um elemento.
21- list-style: define o estilo da lista (disco, quadrado, decimal, etc.).
22- list-style-type: define o tipo de marcador de lista (círculo, decimal, disco, etc.).
23- list-style-position: define a posição do marcador de lista (dentro, fora).
24- background-image: define uma imagem de fundo dentro de um elemento.
25- background-repeat: define como a imagem de fundo deve ser repetida dentro de um elemento (repetir, não repetir, etc.).
26- background-position: define a posição da imagem de fundo dentro de um elemento.
27- background-size: define o tamanho da imagem de fundo dentro de um elemento.
28- opacity: define a transparência de um elemento.
29- z-index: define a ordem de empilhamento dos elementos em relação aos outros elementos.
30- border-color: define a cor da borda de um elemento. Pode ser definido usando nomes de cores, códigos hexadecimais, nomes de cores ou RGB.
31- outline-color: define a cor da linha de contorno de um elemento. Também é comumente usado com valores hexadecimais, nomes de cores ou RGB.
32- box-shadow: define a sombra de um elemento. A cor é um dos valores que podem ser definidos juntamente com outros atributos, como offset, blur e spread.
33- font-style: define o estilo da fonte usada para o texto dentro de um elemento. Os valores comuns são "normal" (padrão) e "italic" (itálico), mas também é possível usar "oblique" (inclinado).
